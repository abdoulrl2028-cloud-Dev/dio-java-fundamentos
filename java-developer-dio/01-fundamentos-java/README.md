# 01 - Fundamentos Java

Conteúdo:

- `sintaxe-java/` - exemplos básicos de sintaxe, variáveis, controle de fluxo e métodos.
- `desafio-conta-bancaria/` - implementação simples de uma conta bancária (depositar, sacar, transferir).
- `exercicios/` - exercícios com soluções (Fibonacci, Palíndromo).

Compilação e execução (exemplos sem build tool):

1. Entre na pasta do exemplo, por exemplo `sintaxe-java`.
2. Compile: `javac Main.java` (ou `javac *.java` conforme o exemplo).
3. Execute: `java Main` (ou o nome da classe com `main`).
