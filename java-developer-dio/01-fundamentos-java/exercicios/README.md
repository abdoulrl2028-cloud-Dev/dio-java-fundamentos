# Exercícios - Fundamentos

Exercícios com soluções simples:

- `solucoes/ExFibonacci.java` - calcula sequência de Fibonacci até N (iterativo).
- `solucoes/ExPalindrome.java` - verifica se uma palavra é palíndroma.

Compile e execute:

```bash
cd exercicios/solucoes
javac *.java
java ExFibonacci 10
java ExPalindrome olá
```
